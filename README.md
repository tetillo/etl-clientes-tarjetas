# Proyecto ETL – Clientes y Tarjetas

## Objetivo
Desarrollar un proceso ETL que lea ficheros CSV de clientes y tarjetas,
realice limpieza, validaciones y anonimización de datos sensibles,
y cargue la información en una base de datos relacional PostgreSQL.

## Tecnologías
- Python 3
- Pandas
- PostgreSQL
- SQLAlchemy
- Git & GitHub
